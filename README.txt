веб-приложение для извлечения метаданных mage
Обзор
Это веб-приложение позволяет пользователям загружать изображения с помощью перетаскивания или выбора файла. Оно извлекает и отображает информацию о метаданных, такую как имя файла, размеры, DPI, глубина цвета и тип сжатия.

Используемые библиотеки
Flask - облегченный веб-фреймворк, используемый для создания веб-приложения.
Pillow - форк библиотеки изображений Python (PIL), который позволяет открывать файлы изображений, управлять ими и сохранять.
Процесс извлечения данных
Загрузка изображений:
Пользователи могут загружать изображения, перетаскивая их в указанную область или выбирая файлы с помощью ввода файла.
Приложение поддерживает одновременную загрузку нескольких файлов.
Обработка изображений:
Загруженные изображения временно сохраняются в каталоге загрузки.
Библиотека Pillow используется для открытия каждого изображения и извлечения метаданных:
Имя файла: Извлечено с помощью метода os.path.basename.
Размеры: Получены с помощью атрибута size, который определяет ширину и высоту в пикселях.
DPI (количество точек на дюйм): Получено из изображения.
